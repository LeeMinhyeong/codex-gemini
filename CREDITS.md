# Credits

This plugin was inspired by:

- [`openai/codex-plugin-cc`](https://github.com/openai/codex-plugin-cc), which demonstrates invoking Codex from Claude Code workflows.
- [`thepushkarp/cc-gemini-plugin`](https://github.com/thepushkarp/cc-gemini-plugin), which adapts the same idea for Gemini CLI and includes Codex-oriented usage guidance.

This repository packages a Codex-focused Gemini CLI bridge and marketplace layout for local or public Codex plugin distribution.
